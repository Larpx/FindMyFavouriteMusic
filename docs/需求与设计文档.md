# 基于.NET的音乐品味预测系统 —— 开发计划与需求文档

## 一、项目概述

### 1.1 项目目标

开发一款基于.NET平台的桌面应用程序，能够：
- 扫描用户本地的音乐文件库
- 提取音频的**专业声学特征**（MFCC、频谱特征等）和**深度学习深度特征向量**
- 构建用户的**音乐品味特征向量画像**（基于均值向量）
- 对任意输入的新音乐文件，计算其与用户画像的**相似度分数**，预测用户对该音乐的喜好程度

### 1.2 核心功能模块

| 模块 | 功能描述 |
|------|----------|
| 音频扫描与加载 | 扫描指定目录，支持多种音频格式（MP3、WAV、FLAC、M4A等） |
| 音频特征提取 | 提取MFCC、频谱质心、色度特征、节奏等专业特征 |
| 深度特征提取 | 使用深度学习模型提取音频的深度特征向量 |
| 用户画像构建 | 基于用户标记为“喜欢”的歌曲特征，计算均值向量 |
| 相似度预测 | 计算新歌与用户画像的余弦相似度/欧氏距离，输出喜好分数 |
| 模型训练（进阶） | 支持训练分类模型（逻辑回归、SVM等）进行喜好预测 |


## 二、技术选型

### 2.1 .NET版本与框架

- **目标框架**：.NET 8.0（LTS版本）或 .NET 10.0（LTS版本）
- **应用程序类型**：Avalonia 桌面应用程序,支持跨平台
- **架构模式**：分层架构（UI层 → 业务逻辑层 → 数据访问层）

### 2.2 核心NuGet包选型

#### 音频处理与特征提取（专业声学特征）

| NuGet包 | 用途 | 说明 |
|---------|------|------|
| **NWaves** | 音频信号处理与特征提取 | .NET生态中最全面的音频DSP库，支持FFT、STFT、MFCC、频谱特征、音高分析等 |
| **NAudio** | 音频文件解码与基础处理 | 支持MP3、WAV、FLAC等多种格式的读取和解析 |

#### 深度学习与深度特征提取

| NuGet包 | 用途 | 说明 |
|---------|------|------|
| **ML.NET** | 机器学习模型训练与推理 | 微软官方跨平台ML框架，支持自定义模型训练和ONNX模型推理 |
| **TensorFlow.NET** 或 **TensorFlowSharp** | TensorFlow模型运行 | 在.NET中运行预训练的TensorFlow音频模型 |
| **TorchSharp** | PyTorch模型运行 | 在.NET中运行预训练的PyTorch音频模型 |
| **Microsoft.ML.OnnxRuntime** | ONNX模型推理 | 高效运行ONNX格式的深度学习模型 |

#### 相似度计算与向量操作

| NuGet包 | 用途 |
|---------|------|
| **MathNet.Numerics** | 向量运算、矩阵运算、余弦相似度计算 |

#### 音频指纹（可选，用于去重）

| NuGet包 | 用途 |
|---------|------|
| **SoundFingerprinting** | 音频指纹提取与识别，可用于去重和相似歌曲检测 |

#### 相似度计算公式

```
专业特征相似度 = CosineSimilarity(新歌专业特征向量, 用户专业特征均值向量)
深度特征相似度 = CosineSimilarity(新歌深度特征向量, 用户深度特征均值向量)
综合相似度 = α × 专业特征相似度 + (1-α) × 深度特征相似度
喜好分数 = 综合相似度 × 100
```

其中 α 为权重系数（建议初始值0.4，可根据效果调整）。

## 五、数据存储设计

### 5.1 数据库表结构（SQLite）

```sql
-- 歌曲表
CREATE TABLE Songs (
    Id TEXT PRIMARY KEY,
    FilePath TEXT UNIQUE NOT NULL,
    FileName TEXT NOT NULL,
    Artist TEXT,
    Album TEXT,
    Genre TEXT,
    Duration REAL,
    FileSize INTEGER,
    BitRate INTEGER,
    SampleRate INTEGER,
    IsLiked BOOLEAN DEFAULT 0,
    CreatedAt DATETIME DEFAULT CURRENT_TIMESTAMP,
    UpdatedAt DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 特征向量表
CREATE TABLE FeatureVectors (
    Id TEXT PRIMARY KEY,
    SongId TEXT NOT NULL,
    FeatureType TEXT NOT NULL,  -- 'acoustic' 或 'deep'
    VectorData BLOB NOT NULL,   -- 二进制存储浮点数组
    VectorDimension INTEGER NOT NULL,
    CreatedAt DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (SongId) REFERENCES Songs(Id) ON DELETE CASCADE
);

-- 用户画像表
CREATE TABLE UserProfiles (
    Id TEXT PRIMARY KEY,
    LikedSongsCount INTEGER DEFAULT 0,
    AcousticMeanVector BLOB,
    DeepMeanVector BLOB,
    CombinedMeanVector BLOB,
    FeatureStdDev BLOB,
    CreatedAt DATETIME DEFAULT CURRENT_TIMESTAMP,
    UpdatedAt DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 预测历史表
CREATE TABLE PredictionHistory (
    Id TEXT PRIMARY KEY,
    SongId TEXT NOT NULL,
    Score REAL NOT NULL,
    Confidence REAL,
    PredictedAt DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (SongId) REFERENCES Songs(Id) ON DELETE CASCADE
);

-- 模型元数据表
CREATE TABLE ModelMetadata (
    Id TEXT PRIMARY KEY,
    ModelName TEXT NOT NULL,
    ModelType TEXT NOT NULL,  -- 'mlnet' 或 'onnx'
    ModelPath TEXT NOT NULL,
    Accuracy REAL,
    TrainingDate DATETIME,
    FeatureCount INTEGER
```

### 6.3 余弦相似度计算

```csharp
using MathNet.Numerics.LinearAlgebra;

public static float CosineSimilarity(float[] vectorA, float[] vectorB)
{
    if (vectorA.Length != vectorB.Length)
        throw new ArgumentException("向量维度必须相同");
    
    var vA = Vector<float>.Build.Dense(vectorA);
    var vB = Vector<float>.Build.Dense(vectorB);
    
    float dotProduct = vA.DotProduct(vB);
    float normA = vA.L2Norm();
    float normB = vB.L2Norm();
    
    if (normA == 0 || normB == 0) return 0;
    
    return dotProduct / (normA * normB);
}
```

### 6.4 ONNX模型推理（深度特征提取）

```csharp
using Microsoft.ML.OnnxRuntime;
using Microsoft.ML.OnnxRuntime.Tensors;

public float[] ExtractDeepFeatures(float[] audioSamples, int sampleRate)
{
    // 1. 生成梅尔频谱图
    float[,] melSpectrogram = GenerateMelSpectrogram(audioSamples, sampleRate);
    
    // 2. 转换为模型输入张量 [1, 1, 高度, 宽度] 或 [1, 3, 224, 224]
    var inputTensor = new DenseTensor<float>(melSpectrogram.Cast<float>().ToArray(), 
        new[] { 1, 1, melSpectrogram.GetLength(0), melSpectrogram.GetLength(1) });
    
    // 3. 运行ONNX模型推理
    using var session = new InferenceSession("vggish.onnx");
    var inputs = new List<NamedOnnxValue>
    {
        NamedOnnxValue.CreateFromTensor("input", inputTensor)
    };
    
    using var results = session.Run(inputs);
    var output = results.First().AsTensor<float>();
    
    // 4. 返回深度特征向量
    return output.ToArray();
}
```
